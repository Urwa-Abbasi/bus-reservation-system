# Bus Reservation System

A web-based bus ticket booking system with seat selection and admin management.

### Features:
- Route search and bus selection
- Interactive seat booking system
- Admin panel to add/manage buses
- Booking confirmation and history
- User login/signup

### Tech Stack:
HTML, CSS, JavaScript, PHP, MySQL


### How to run:
1. Clone this repository
2. Import database.sql in phpMyAdmin
3. Update database config in config.php
4. Run on local server: XAMPP/WAMP
5. Open localhost/bus-reservation-system in browser
